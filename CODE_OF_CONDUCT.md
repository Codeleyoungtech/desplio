# Code of Conduct

## Our Standard

We want Desplio to be a respectful, technically generous project where people can collaborate safely.

Examples of positive behavior:

- being constructive and specific
- welcoming questions
- assuming good intent while staying honest
- giving feedback on code and ideas without personal attacks

Unacceptable behavior includes:

- harassment or intimidation
- discriminatory language
- bad-faith disruption
- doxxing or sharing private information

## Enforcement

Project maintainers may remove comments, issues, pull requests, or participation that violate this code of conduct.

## Scope

This applies within project spaces and in public interactions representing the project.
